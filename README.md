# tms-min-side-bunn
Mikrofrontend som trekkes inn av Min Side.

## Kom i gang
1. Bygg tms-min-side-bunn ved å kjøre npm run build
2. Start appen lokalt ved å kjøre npm start
3. Appen nås på http://localhost:4300
